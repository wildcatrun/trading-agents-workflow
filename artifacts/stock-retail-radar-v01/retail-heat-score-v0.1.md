## 1. RetailHeatScore 核心指标清单

RetailHeatScore 用 0-100 分表示散户活跃度强弱和变化速度，建议由以下指标组成：

1. 讨论量：帖子数、评论/回复数、独立发帖人数。
2. 增长斜率：1 日、3 日、5 日环比，偏离 20 日基线幅度。
3. 热榜曝光：是否进入股吧、雪球、平台热榜，停留时长与排名变化。
4. 搜索热度：关键词搜索指数及短期加速度。
5. 互动强度：评论率、转发率、收藏率、单帖平均互动。
6. 情绪倾向：看多、看空、中性比例。
7. 极端表达密度：涨停、翻倍、抄底、梭哈、崩了等极端词频占比。
8. 分歧度：多空同时放大、观点集中度下降。
9. 持续性：高热是否连续 2-5 个观察窗，而非单点脉冲。

初始口径：

- 以“讨论量 + 增长斜率 + 搜索热度 + 互动强度”为主骨架。
- 以“情绪倾向 + 极端表达 + 分歧度”为修正项。
- 缺失数据只降低置信度，不直接判低热度。

## 2. 数据源优先级

### P1 高优先级

- 股吧、雪球等股票讨论区的帖子数、评论数、活跃用户数。
- 平台热榜、热门榜、飙升榜。
- 搜索指数类数据。
- 看多/看空投票或平台自带情绪比例。

### P2 中优先级

- 微博、公开社区、短内容平台等社交平台情绪讨论。
- 评论增长率、转发增长率。
- 题材联动讨论热度和同概念扩散。

### P3 辅助数据

- 非结构化情绪词云。
- KOL 单帖带动热度。
- 二手聚合榜单、媒体转述热度。

优先原则：先用稳定、可重复抓取、可审计平台；先聚合指标，禁止无边界保存帖子全文；搜索热度与讨论热度分开记录，不能互相替代。

## 3. SentimentStage 判定规则

| stage | 初始判定 |
| --- | --- |
| cold | RetailHeatScore < 25，且 5 日均值附近无明显抬升 |
| warming | 25-50，且 3 日/5 日增速为正，开始脱离 20 日基线 |
| hot | 50-75，且讨论量、搜索热度、互动强度至少两项同步上升 |
| euphoric | >=75，且极端表达密度高、热榜持续、看多单边化明显 |
| cooling | 从 hot/euphoric 回落，连续 2 个观察窗下降，或热度仍高但互动与搜索先转弱 |

补充：cooling 不等于 bearish，只表示情绪退潮阶段；stage 必须由“水平 + 变化”共同判断，不能只看单日绝对值。

## 4. DisagreementScore 与 EmotionRiskFlags

### DisagreementScore

DisagreementScore 表示市场观点分裂程度，0-100 分。初始由以下因素组成：

- 看多/看空比例接近 50:50 时更高。
- 同时出现高热度与高反驳密度时更高。
- 主题扩散但结论不一致时更高。
- 单边一致预期时反而下降。

### EmotionRiskFlags

- `crowding_spike`：讨论量/搜索热度短期异常飙升。
- `euphoric_language`：极端乐观词频异常。
- `panic_dump`：极端悲观/恐慌词频异常。
- `one_sided_bullish`：看多比例过度单边。
- `one_sided_bearish`：看空比例过度单边。
- `narrative_overprice`：叙事强、证据弱，需猫之眼/猫之耳交叉确认。
- `fade_after_spike`：热度冲高后快速衰减。
- `rumor_driven_heat`：热度主要由未经证实传闻驱动。

## 5. 亮区、过热区、暗区的升级观察触发

### 亮区

触发：RetailHeatScore >= 70 且至少连续 2 个观察窗维持高位，同时搜索热度抬升或进入热榜。

动作：升级为高频观察；请猫之眼核查催化，请猫之耳核查基本面支撑。

### 过热区

触发：RetailHeatScore >= 85，且 `euphoric_language` 或 `one_sided_bullish` 触发；若 `fade_after_spike` / `rumor_driven_heat` 并存，优先标红。

动作：升级风险观察，不生成交易结论，只提示拥挤与反身性风险上升。

### 暗区

触发：RetailHeatScore <= 35，但出现 warming 早期迹象；或低热度下搜索/讨论开始边际改善。

动作：升级为中频研究候选，但必须联动猫之眼/猫之耳确认是否真有早期变化，避免把冷清误当机会。

## 6. 禁止误用规则

1. 热度不等于买入，低热度也不等于低风险。
2. 散户活跃度只能描述关注变化，不能单独推出上涨/下跌结论。
3. 数据缺失、抓取失败、历史太短，禁止解释为“没人关注”。
4. 单个平台爆量，不能直接代表全市场情绪。
5. 情绪过热不是做空信号；情绪冷清也不是潜伏信号。
6. 未经猫之眼、猫之耳交叉验证，猫之鼻不得把叙事热度包装成基本面改善。
